## リクルート：ポケモンの入手方法

本作では、従来の「ポケモンゲット」のループは「リクルート」と呼ばれるシステムに置き換えられています。野生でモンスターボールを使うことはありません。その代わりに、日替わりのラインナップ、または外部ストレージからの転送によってポケモンを入手します。

22時間ごとに、ゲーム内ではステータス、特性、技がランダムに設定された10匹のポケモンが提示されます。これらのポケモンは2つのステータスがすでに最大まで育成された状態で提供されますが、その配分は最適ではないことが多く、対戦で通用させるにはさらなる投資が必要になります。

### リクルートタイプの比較

| 特徴 | お試しリクルート | 恒久リクルート |
| :--- | :--- | :--- |
| **コスト** | 0 VP (無料) | 2,500 VP または チームメイトチケット1枚 |
| **期間** | 7日間 | 無制限 |
| **カスタマイズ** | 再育成や技変更不可 | フルカスタマイズ可能 |
| **アップグレード** | 恒久リクルートへ昇格可能 | 該当なし |
| **主な用途** | 新しいチーム編成のテスト | 対戦の主力メンバーの構築 |

> **💡 ヒント:** 特定のポケモンがチームのシナジーに合うかどうかを確認するために、2,500 VPを支払って恒久リクルートする前に「お試しリクルート」を活用しましょう。

特定のポケモンを探していて日替わりラインナップにいない場合は、**クイッククーポン**を使用してラインナップを更新できます。クーポン1枚でタイマーが1時間短縮されるほか、2,200 VPを消費して10匹のラインナップを即座にフルリセットすることも可能です。

## Pokemon Homeとの連携とボックス容量

**ポケモンチャンピオンズのメカニクス**を活用する最も効率的な方法は、Pokemon Homeを経由することです。Homeからポケモンを転送するのは無料で、リクルートに伴うVPコストを回避できます。これは、『ポケットモンスター スカーレット・バイオレット』や『ソード・シールド』ですでに対戦準備の整ったポケモンを所有しているベテラントレーナーにとって特に価値があります。

Homeから連れてこられたポケモンは「訪問中」という扱いになります。そのポケモンはHomeのストレージに残ったままゲーム内でのバトルに使用可能になります。ゲーム内で行った育成や技の変更は、Homeにいる元のポケモンには影響しないため、自身のコレクションに永続的な変化を与えずに試行錯誤することができます。

### ボックス容量とストレージ制限

| パッケージタイプ | ボックス容量上限 | 追加特典 |
| :--- | :--- | :--- |
| **無料版** | 30スロット | 基本アクセス |
| **スターターパック ($6.99)** | 80スロット | 育成チケット50枚、チームメイトチケット30枚 |
| **メンバーシップ ($4.99/月)** | 1,000スロット | 限定ミッション、バトルチーム枠の増加 |

> **⚠️ 注意:** ゲーム内で直接リクルートしたポケモンやギフトとして受け取ったポケモンは、Pokemon Homeに転出させることはできません。Homeから連れてきたポケモンのみが戻すことができます。

## 育成と最適化のコスト

ポケモンをリクルートしたら、次は最適化です。**ポケモンチャンピオンズのメカニクス**は、従来の育成の常識を覆しました。以前のゲームでは、努力値（ステータス）の育成が最も時間を要し、技の変更は容易でした。しかし2026年現在、技の変更こそがプロセスの中で最もコストのかかる部分となっています。

### ビクトリーポイント（VP）コスト表

| 育成アクション | VPコスト | 代替手段 |
| :--- | :--- | :--- |
| **ステータス1点上昇** | 5 VP | 育成チケット |
| **全ステータス育成（66点）** | 330 VP | 育成チケット |
| **性格変更** | 500 VP | 育成チケット |
| **特性変更** | 500 VP | 育成チケット |
| **技を1つ変更** | 250 VP | 育成チケット |
| **技を4つすべて変更** | 1,000 VP | 育成チケット |

技の変更には1つにつき250 VPかかるため、あらかじめ希望の技構成と特性を持ったポケモンを [Pokemon Home](https://home.pokemon.com/en-us/) から転送することを強くお勧めします。これにより、VPを必須の持ち物やメガストーンの購入のために節約できます。

## バトルフォーマットとランクマッチ

対戦こそがこのゲーム体験の核心です。他のプレイヤーと対戦する方法は主に、ランクバトル、カジュアルバトル、プライベートバトル、インターネット大会の4つがあります。

### ランクティアシステム

ランクバトルはティア（階級）に分かれており、各ティアには4つのステージ（例：モンスターボール4〜1）があります。これらのティアを進めることが、VPやシーズン報酬を獲得する主な手段です。

1.  **ビギナー級:** 新規プレイヤー向けのチュートリアルフェーズ。
2.  **モンスターボール級:** 標準的なラダーの開始点。
3.  **スーパーボール級:** 難易度が上がり、報酬も良くなります。
4.  **ハイパーボール級:** メタ戦略が必須となるハイレベルな戦い。
5.  **マスターボール級:** トップクラスのトレーナーが集うエリート層。
6.  **チャンピオン級:** マスターボールの上位に位置する、最高峰のプレイヤーのための新しい名誉あるティア。

### 対戦形式

現在、ゲームでは2つの主な対戦スタイルがサポートされています。以前のカジュアルバトルで定番だった6vs6のシングルバトルは、現在すべてのモードで利用できないことに注意してください。

*   **ダブルバトル:** 公式の対戦フォーマット。6匹のポケモンを連れて行き、その中から4匹を選んで4vs4で戦います。
*   **シングルバトル:** 3vs3のフォーマット。6匹のポケモンを連れて行き、3匹を選んで戦います。

## 経済と収益化

このゲームは、主要通貨であるVPに対して「プレイして稼ぐ（play-to-earn）」モデルを採用しています。現実の通貨でVPを直接購入することはできませんが、時間を節約するためのリソースが含まれた「スターターパック」や「メンバーシップ」を購入することは可能です。

### デイリーおよびウィークリー収入
VPの蓄えを維持するために、毎日以下の項目を完了することを目指しましょう：
*   **ログイン報酬:** シンプルな毎日のチェックイン。
*   **リクルート:** 少なくとも1匹のポケモンをリクルート（お試しまたは恒久）。
*   **ランクマッチ:** 1回のランクバトルを完了（勝敗は問いません）。
*   **ミッション:** 「効果抜群の技を3回使う」や「くさタイプで勝利する」といったデイリー/ウィークリー目標を達成。

これらのタスクを完了することで、通常1日あたり約500 VPと24枚のクイッククーポンを獲得できます。

### ゲーム内ショップ
VPはすべての対戦用アイテムの購入に使用されます。レギュレーションM-Aではアイテムプールが限られているため、定番アイテムを揃えておくことが重要です。

*   **きのみ:** 400 VP
*   **持ち物（いのちのたま、こだわりメガネなど）:** 700 - 1,000 VP
*   **メガストーン:** 2,000 VP
*   **バトルBGM:** 5,000 VP

## FAQ

**Q: ポケモンチャンピオンズのメカニクスでVPを最も速く稼ぐ方法は何ですか？**

A: 最も確実な方法は、デイリーおよびウィークリーミッションを完了し、ランクバトルに参加することです。ランクバトルでは勝敗に関係なくVPを獲得できますが、勝利したほうが報酬は大幅に多くなります。

**Q: 以前の作品のようにレンタルチームは使えますか？**

A: システムが「レプリカチーム」に変更されました。コードを入力するだけで無料でチームを使えた以前のレンタルシステムとは異なり、レプリカチームでは対象の6匹のポケモンと必要なアイテムを実際に所有している必要があります。レプリカコードを使用すると、手持ちのポケモンが自動的にチームのステータスや技に合わせて育成されますが、その際もVPや育成チケットを消費します。

**Q: メンバーシップは月額料金を払う価値がありますか？**

A: プレイスタイルによります。メンバーシップは主にパワーユーザー向けで、1,000個のボックス枠、15個のバトルチーム枠、そしてチームメイトチケットや育成チケットが手に入る限定ミッションを提供します。1つか2つのチームしか使わない予定であれば、無料版や一度きりのスターターパックで十分でしょう。

**Q: ポケモンチャンピオンズでメガシンカは使えますか？**

A: はい、メガシンカは現在のメタの核心部分です。メガストーンはショップで各2,000 VPで購入するか、シーズンのバトルパスを通じて獲得できます。各ポケモンがバトル中にメガシンカするには、それぞれの専用ストーンが必要です。

## 同期のためのアカウント準備

ポケモンの移動を開始する前に、ハードウェアとソフトウェアが適切にアップデートされていることを確認してください。2026年4月現在、この連携は従来のNintendo Switchと新型のNintendo Switch 2の両方をサポートしています。このプロセスはすべて、クラウドストレージとローカルのセーブデータを結ぶ架け橋となる「ニンテンドーアカウント」に依存しています。

まず、両方のアプリケーションで最新のアップデートをダウンロードしていることを確認してください。転送が一方通行であることが多かった過去の作品とは異なり、このシステムでは「おでかけ（Visiting）」という仕組みを採用しています。これは、ポケモンが恒久的に移動するのではなく、新しいゲーム環境に「貸し出されている」状態であることを意味します。

| 必要事項 | 詳細 |
| :--- | :--- |
| **ニンテンドーアカウント** | 両方のゲームで使用しているアカウントが同一であること。 |
| **加入プラン** | 『Pokémon HOME』のフリープランまたはプレミアムプラン（どちらでも可）。 |
| **ゲームバージョン** | 『Pokémon Champions』バージョン 1.2.0 以上。 |
| **ハードウェア** | **Nintendo Switch** または **Nintendo Switch 2**。 |

> **注意:** これらの転送を行うために『Pokémon Champions』の有料サブスクリプションや特定のVP（ビクトリーポイント）残高は必要ありません。このサービスはすべてのプレイヤーに完全に無料で提供されます。

## 『Pokémon Champions』と『Pokémon HOME』を連携する方法

初期設定は『Pokémon HOME』のインターフェース内で行います。コンソールでアプリケーションを起動すると、最新作の牧場型ゲームプレイ専用の新しいアイコンが表示されていることに気づくはずです。

1.  『Pokémon HOME』を起動し、メイン画面から「ポケモン」メニューを選択します。
2.  『Pokémon Champions』のアイコンを探します（『スカーレット・バイオレット』や『Pokémon GO』などの他の対応タイトルと並んで表示されます）。
3.  システムから、本体の特定のユーザーアカウントに接続するよう求められます。選択を確定してください。
4.  連携が完了すると、ストレージボックスが表示されます。これで、HOMEのボックスと牧場用の「おでかけ」スロットを切り替えることができるようになります。

## 「おでかけさせる」プロセスの開始

ポケモンを移動させる際、単にデータを動かしているわけではありません。「おでかけ（Visit）」を開始しているのです。これは2026年のエコシステムにおける重要な区別です。ポケモンが牧場におでかけしている間、クラウドストレージ内でのそのポケモンのステータスはロックされます。

ポケモンを送る手順：
*   『Pokémon HOME』のボックスに移動します。
*   転送したい特定のポケモンを選択します。
*   **Yボタン**を押して「おでかけさせる」を選択します。
*   おでかけ期間中は、特定の操作（逃がす、他のゲームへ送るなど）が無効になることを説明する警告プロンプトを確認します。

### 転送の制限とステータス

| アクション | おでかけ中のステータス |
| :--- | :--- |
| **他のゲームへの転送** | **不可** |
| **ボックスから逃がす** | **不可** |
| **マークをつける** | **不可** |
| **HOMEでの交換** | **不可** |
| **Championsでの対戦使用** | **可能** |

## 『Pokémon Champions』でポケモンを受け取る

クラウドで転送を開始したら、ゲーム側で手動でゲストを「受け取る」必要があります。これは牧場のメインメニュー内にある勧誘（Recruit）インターフェースで行います。

1.  『Pokémon Champions』を開き、**「勧誘（Recruit）」**メニューに移動します。
2.  **「Pokémon HOMEから受け取る」**オプションを選択します。
3.  ゲームが、お使いのニンテンドーアカウントに紐付けられた待機中の訪問者をチェックします。
4.  受け取りを確定します。すると、ポケモンがアクティブボックスに表示されます。

> **💡 ヒント:** ポケモンが初めておでかけしてきた際、その「現在のつよさ（Current Strength）」は以前のステータスに基づいて計算されます。ただし、本作では努力値（EV）や個体値（IV）の機能が異なるため、現在のメタに適合した新しい「つよさ」の値が割り当てられます。

## 技とステータス変更の管理

**『Pokémon Champions』と『Pokémon HOME』の連携**システムに関して最も多い質問の一つが、技構成と対戦での実用性についてです。2026年のアップデート以降、技は新しいゲームの内部データベースに存在する限り、一般的に維持されます。もしポケモンがサポートされていない技を覚えている場合、クラウドに戻るまで一時的に別の技に置き換わるか、使用不可になることがあります。

| 特徴 | Champions入局時の挙動 |
| :--- | :--- |
| **性格** | **変更なし** |
| **技** | **維持**（データベースに存在する場合） |
| **特性** | **変更なし** |
| **努力値/個体値** | **「つよさ」ポイントに変換** |
| **育成データ** | **セーブデータにローカル保存** |

牧場環境内で行われたトレーニングは特殊です。おでかけ中にポケモンのパフォーマンスを向上させることはできますが、これらの特定の「つよさ」の向上によって、クラウドストレージ内の元の努力値/個体値データが上書きされることはありません。ただし、同じポケモンが将来再びおでかけしてきた場合、フォルムチェンジなどをしていない限り、ゲームは以前のトレーニングの進捗を記憶しています。

## ポケモンを『Pokémon HOME』に戻す方法

ポケモンを戻すのも連れてくるのと同様に簡単で、引き続き無料のサービスです。これは、特定のリボンやイベントのために、ポケモンを『ポケットモンスター スカーレット・バイオレット』などの本編タイトルに戻す必要がある場合に特に便利です。

1.  『Pokémon Champions』の**「勧誘（Recruit）」**メニューに移動します。
2.  **「Pokémon HOMEにポケモンを戻す」**を選択します。
3.  おでかけ中のポケモンを選択し、**Xボタン**を押します。
4.  戻すことを確定します。ポケモンは即座に牧場から消え、クラウドボックスに再表示されます。

クラウドアプリケーションに再度ログインすると、ゲームアイコンに**赤いドット**のインジケーターが表示され、訪問者が戻ってきたことを知らせてくれます。そのポケモンの能力画面には、Championsのサーキットに参加したことを証明する特別なシンボルが付与されます。

## よくある連携トラブルの解決方法

接続の確立に問題がある場合は、チェックすべき一般的な障害がいくつかあります。ほとんどの問題は、アカウントの不一致やリージョンロックに起因します。

*   **アカウントの不一致:** コンソールにログインしているニンテンドーアカウントが、クラウドサブスクリプションを保持しているアカウントと同じであることを確認してください。
*   **ロード時間が長い:** 両方のアプリケーションは[ポケモン公式サイトのサーバー](https://www.pokemon.com/us/pokemon-video-games/pokemon-home/)と常に通信を行う必要があります。サーバーに負荷がかかっている場合、「ニンテンドーアカウントを確認中」の画面で止まることがあります。
*   **フォルムチェンジ:** 他のゲームでポケモンのフォルム（メガシンカや特定のリージョンフォームなど）を変更した場合、牧場に戻った際に以前のトレーニングデータを認識できない場合があります。

## FAQ

**Q: 『Pokémon Champions』と『Pokémon HOME』の連携機能を使うのにVPは必要ですか？**

A: いいえ、転送プロセスは完全に無料です。クラウドからポケモンを送受信するために、ゲーム内通貨やビクトリーポイントを消費する必要はありません。

**Q: おでかけした後、ポケモンのステータスは永久に変わってしまいますか？**

A: いいえ。ゲーム内では「つよさ」を決定するために異なるシステムを使用していますが、元の努力値や個体値はクラウドに保存されています。おでかけ中に行われたトレーニングは牧場のセーブデータ内に留まり、他のゲームには引き継がれません。

**Q: 『Pokémon GO』から『Champions』に直接ポケモンを送ることはできますか？**

A: 直接はできません。まず『Pokémon GO』から『Pokémon HOME』にポケモンを送り、そこから牧場へのおでかけを開始する必要があります。

**Q: ポケモンがおでかけしている間にセーブデータを失ったらどうなりますか？**

A: ポケモンはおでかけ中、技術的にクラウド上で「ロック」されているため、ローカルのセーブデータを失うと問題が発生する可能性があります。おでかけ中のポケモンへのアクセスを失わないよう、常にNintendo Switch Onlineを通じてクラウドセーブデータがバックアップされていることを確認してください。
